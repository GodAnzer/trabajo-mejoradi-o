# Glosario técnico

| Concepto | Definición |
|---|---|
| HTML | Lenguaje que estructura el contenido de una página web. |
| CSS | Lenguaje que define estilos visuales: colores, tamaños, distribución y responsive. |
| JavaScript | Lenguaje que permite interacción y lógica en el navegador. |
| Formulario | Conjunto de campos que recoge datos del usuario. |
| Validación | Proceso de comprobar si un dato cumple reglas. |
| Integridad de datos | Calidad de los datos para que sean correctos, completos, coherentes y útiles. |
| UX | Experiencia de usuario: facilidad y claridad al usar un sistema. |
| localStorage | Almacenamiento del navegador que conserva datos localmente. |
| Evento | Acción que ocurre en la página, como escribir, hacer clic o enviar un formulario. |
| DOM | Representación de la página que JavaScript puede leer y modificar. |
| Depuración | Proceso de encontrar, analizar y corregir errores. |
| Caso de prueba | Situación diseñada para comprobar si el sistema funciona correctamente. |
| Duplicado | Registro repetido que afecta la calidad de la información. |
| Responsive | Capacidad de adaptarse a diferentes tamaños de pantalla. |
| Accesibilidad | Diseño que permite que más personas puedan usar la aplicación. |
